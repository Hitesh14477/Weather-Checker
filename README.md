# 🌦️ Weather Checker

A simple and responsive weather application that allows users to search for any city and view real-time weather information such as temperature, humidity, and weather conditions.

## 🚀 Features
- Search weather by city name
- Real-time weather data
- Displays temperature and weather conditions
- Clean and responsive UI
- Error handling for invalid cities

## 🛠️ Tech Stack
- React.js
- JavaScript
- CSS / Tailwind CSS
- OpenWeatherMap API


## 🔑 API Used
This project uses the **OpenWeatherMap API** to fetch weather data.

https://openweathermap.org/api
